# ACT Score Analysis

This project analyzes student ACT math performance and investigates how low-effort test-taking behavior impacts overall performance metrics.

## Key Findings
- Approximately 12% of students were identified as low-effort based on rapid completion time and low performance.
- These students lowered the overall average math percentile by over 5 points.
- Time spent on the test showed a moderate positive correlation with performance (r ≈ 0.44).

## Notes
- The dataset is not included due to student privacy and FERPA considerations.
- All personally identifiable information (PII) was removed prior to analysis.
